# Penzion na Krétě

Web penzionu (Pardubice) – statické HTML/CSS/JS s administrací fotogalerie a ceníku přes Supabase.

## Stránky
- `index.html` – Domů
- `cenik.html` – Ceník (editovatelný z adminu)
- `fotogalerie.html` – Fotogalerie (editovatelná z adminu)
- `kontakt.html` – Kontakt (formulář přes Formspree)
- `admin.html` – Administrace (heslo, fotky, ceník, změna hesla)

## Nastavení
Konfigurace Supabase je v `config.js`. Podrobný návod v `NAVOD.md`.
